Initialize minimum amount required before checkout.

== Description ==
Disables all checkout options when cart has not met minimum amount set through this plugin.

== Installation ==
= Automatic installation =
1. log in to your WordPress dashboard, go to the Plugins menu and click Add New.
2. Search for “Minimum price required to checkout for WooCommerce”.
3. Once found, install by clicking “Install Now” button.
4. Click on activate after installation.
5. A menu will appear under Settings main menu after activation( Minimum price required to checkout for WooCommerce ).

= Manual installation =
1. Download the plugin(zip format).
2. Go to Plugin dashboard, Add new and upload the zip folder.
3. Activate the plugin after unzipping/installation.
5. A menu will appear under Settings main menu after activation( Minimum price required to checkout for WooCommerce ).
